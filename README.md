# Clustering & Better Visualizations

## Objective: 
Clustering of data & better visualizations using Python.

## Tools:
Python - 
  * Dataframe - pandas
  * Clustering - sklearn: KMeans clustering
  * Visualization - matplotlib, seaborn, voronoi
  * Geo distance - geopy
